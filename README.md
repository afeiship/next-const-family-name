## installation
```bash
npm install -S @jswork/next-const-family-name
```

## usage
```js
import '@jswork/next-const-family-name';

const names = nx.constFamilyName; // [''赵', '钱', '孙', '李', '周', '都', '吴', '郑',.....]
```
